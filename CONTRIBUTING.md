# Contributing

This repository hosts **technical documentation** for beesi.ai — architecture, workflows, and integration notes.

## How to contribute

1. **Small fixes** — Typos, broken links, clearer wording: open a pull request.
2. **Substantive changes** — If your edit reflects new product behavior, ensure it matches **`openapi.json`** and deployed reality (or add an explicit “pending rollout” note).
3. **Diagrams** — Prefer **Mermaid** in Markdown so changes stay reviewable in Git.

## Language

**English** is the primary language for files in this repo. Optional translations can live alongside as `*.zh-CN.md` (or similar) if you add them later.
